Qu'est ce qu’une veille technologique ?

C'est une activité consistant à se tenir informé des innovations dans un secteur déterminé.

Qu'est ce que Node.JS ?

Node.js est un interpréteur de JavaScript qui permet d'exécuter du code JavaScript de manière serveur.
Il a été créé par Ryan Dahl en 2009 et est maintenu par le projet Node.js.
Node.js utilise le modèle d'exécution évènementiel (event-driven) pour traiter les requêtes et permettre une grande performance et flexibilité.
Il est souvent utilisé pour créer des applications web, des APIs, des serveurs et des outils de ligne de commande.

Node.js est actuellement l'un des frameworks les plus populaires pour la création d'applications web et de serveurs, et est largement utilisé dans l'industrie de la technologie.

Les caractéristiques

  • Exécution de code JavaScript serveur

  • Modèle d'exécution événementiel - Asynchrome

  • Utilisation de bibliothèques et frameworks tiers

Projet

  • Node v23.7.0 (Actuel)

    Node.js a publié la version 23.7.0, qui comprend des mises à jour notables telles qu'une option permettant d'exclure les modèles glob dans les opérations du système de fichiers, l'ajout de la prise en charge de TypedArray et DataView dans SQLite et l'introduction d'un nouveau type d'erreur pour la syntaxe TypeScript non prise en charge. Divers modules ont été mis à jour, les exécuteurs de tests reçoivent des fonctionnalités supplémentaires et plusieurs améliorations internes et corrections de bogues sont intégrées à cette version.

  • Node v20.18.1 (LTS)

    La version 20.18.3 de Node.js, dont le nom de code est « Iron », introduit plusieurs changements notables, notamment le marquage des attributs d'importation et du module JSON comme stables, la mise à jour des certificats racines vers NSS 3.104 et l'ajout de nouveaux collaborateurs comme LJHarb et jazelly. Cette version améliore également la documentation en appliquant des politiques plus strictes pour les versions semver-major et améliore divers tests de performance et processus de construction.

  • Node v22.14.0 (LTS)

    La version 22.14.0 (LTS) de Node.js introduit plusieurs nouvelles fonctionnalités et améliorations. Les mises à jour incluent des certificats racine cryptographiques pour NSS 3.107, des améliorations dans les opérations du système de fichiers avec la possibilité d'exclure des modèles et l'ajout de la prise en charge de TypeScript dans les évaluations STDIN. De nouveaux utilitaires et API ont été intégrés, tels que `findPackageJSON`, `process.ref()` et `process.unref()`, ainsi que des améliorations dans le testeur et la prise en charge de SQLite.

  • Node v23.8.0 (Actuel)

    Node.js v23.8.0 introduit plusieurs nouvelles fonctionnalités, notamment la prise en charge des certificats d'autorité de certification système sur macOS et Windows, l'API URL Pattern, l'algorithme de compression zstd et les threads nommés pour un débogage amélioré. Les mises à jour supplémentaires incluent les données de fuseau horaire pour 2025 et les améliorations des fonctions SQLite.
